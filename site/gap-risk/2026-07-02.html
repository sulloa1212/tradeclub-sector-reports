11How To Read This Report

Implied Move (Gap / 1-Week)
The ±1-standard-deviation range derived from each index's own volatility index (VIX/VXN/RVX/VXD). About 2-in-3 moves land inside the band; roughly 1-in-3 exceed it. Wider band = more uncertainty, not necessarily more danger.
Two Horizons
The gap horizon covers only the next opening (Thu close → Mon open for this holiday report). The 1-week horizon covers the next ~5 trading sessions. They use different math: the gap uses an overnight variance factor (0.57) × holiday bump (1.25); the 1-week does not.
Quiet Row (≤0.5% gap / ≤1% week)
The first row in every odds table = the chance the index barely moves (within the smallest threshold). These tiny moves are common; they're listed first so every row in the table sums to ~100% across down + up + quiet.
Risk Dial (Calm / Elevated / High / Extreme)
Blends implied move size with fragility (is the cushion on? is a catalyst inside the window?). For the gap: Calm <0.4%, Elevated 0.4–0.7%, High 0.7–1.05%, Extreme >1.05%. For the week: Calm <1.5%, Elevated 1.5–2.5%, High 2.5–3.5%, Extreme >3.5%. A "High" dial means wider expected range — it is not automatically bearish.
Direction Split (Down vs Up odds)
We take the symmetric probability band and split it using a two-piece skew based on put-skew and gamma regime. The two legs always add up to the symmetric band at each threshold — we're redistributing the probability, not inventing new risk. The lean (e.g. "57% ↓") is the headline skew; it is a tendency, not a prediction.
Big Move Ranking
