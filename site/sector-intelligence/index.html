6How to Read This Report

Here is what each number and label means — the generic method lives here, the results are in the sections above.

Direction Score (−100 to +100)

A single number summarizing a sector's swing-trade outlook. +100 = maximum bullish; −100 = maximum bearish; 0 = neutral / no clean setup.
